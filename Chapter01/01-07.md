# 常常聽到的私醫聯招、臺聯大、臺綜大是什麼？

## 1. 私醫聯招

私醫是所有私立醫學院所共同舉辦的聯合招生，目前有台北醫學大學、中山醫學大學、中國醫藥大學、長庚大學、高雄醫學大學、慈濟大學共六所，未來可能會陸陸續續增加也說不定，這些學校聯招科目為國文、英文、化學、生物。

## 2. 臺灣聯合大學系統（臺聯大）

臺聯大指的是臺灣聯合大學系統，由清華大學、交通大學、中央大學、陽明大學聯合招生，考試科目有分群組，利用志願分發的方式來錄取。**若無特殊狀況，臺聯大系統考試日期往年來皆與國立臺灣大學相撞**。

## 3. 臺灣綜合大學系統（臺綜大）

臺綜大指的是臺灣綜合大學系統，由成功大學、中山大學、中興大學、中正大學於 2012 年組成之校際聯盟，傳言於  2014 年將舉行研究所與轉學考試之聯合招生。（但該年度仍各自獨立招生），最新消息由國立中正大學校長吳志揚於 [2015.09.14 校長給全校師生同仁的一封信](http://president.ccu.edu.tw/) 中提及：「**本校所屬的台灣綜合大學系統轉學考這學期首度辦理聯合招生、統一分發。**」

## 4. 其他

除此之外，淡江理學院、中原理學院以及高屏澎聯招也是利用志願分發的方式聯合招生。
